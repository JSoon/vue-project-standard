# vue-project-standard

Vue.js 项目标准脚手架.

> ⚠️⚠️⚠️ 该项目已废弃, 新项目使用 [vue-project-2.x](https://github.com/JSoon/vue-project-2.x)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```
